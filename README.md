# Webpack Boilerplate

## Install

```
npm install
```

## How to build

```
npm run build
```

## Dependency

```
npm install webpack
```

```
npm install html-webpack-plugin clean-webpack-plugin
```

```
npm install style-loader css-loader node-sass sass-loader
```

```
npm install babel-loader babel-core babel-preset-es2015
```
