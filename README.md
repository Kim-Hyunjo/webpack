# Webpack Boilerplate

## Install

```
git clone https://github.com/Kim-Hyunjo/webpack.git
```

```
npm install
```

## Open develop server

```
npm run dev
```

## Build

```
npm run build
```

## Dependency

- webpack webpack-cli
- style-loader css-loader node-sass sass-loader
- babel-loader babel-core babel-preset-es2015
- html-webpack-plugin
- mini-css-extract-plugin

## Stylesheets

- base.css: css 속성 초기화에 사용
