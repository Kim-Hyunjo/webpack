# Webpack Boilerplate

## Install

```
npm install
```

## How to build

```
npm run build
```

## Dependency

- webpack
- html-webpack-plugin
- clean-webpack-plugin
- style-loader css-loader node-sass sass-loader
- babel-loader babel-core babel-preset-es2015

## Stylesheets

- reset.css: css 속성 초기화에 사용
